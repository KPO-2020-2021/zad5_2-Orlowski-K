Opis szczegółowy

Klasa opisuje pojecie bryly geometrycznej w przestrzeni. Sklada sie z pol przechowujacych sciezki do plikow z wzorcowymi brylami oraz docelowym miejscem skladowania bryl. Wektor jest uzywany do slkalowania bryl.
