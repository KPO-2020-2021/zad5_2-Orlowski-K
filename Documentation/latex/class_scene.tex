\doxysubsection{Opis szczegółowy}
Klasa opisuje pojecie sceny jako przestrzeni, w której znajduja sie zadane obiekty. Moze zawierac wiele obiektow typu \mbox{\hyperlink{class_drone}{Drone}}. Jako pola prywatne zzwiera kontener dronow, sciezke do pliku z plaszczyzna, Lacze do programu G\+N\+U\+Plot i numer aktywnego drona.
